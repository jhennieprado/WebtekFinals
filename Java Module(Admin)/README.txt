If you want to do a request/response trace, check the servlet being called
by index.jsp


1. Download a copy of apache tomcat
2. Download NetBeans
3. Go to the plugins section and then install the "java EE base" and "EAB AND EJB" plugins
4. If prompted to restart netbeans then restart
5. Setup apache tomcat properly (https://www.ntu.edu.sg/home/ehchua/programming/howto/Tomcat_HowTo.html)
5. Import BxB.zip
6. If there are problmes, right click on the project and the choose the
resolve problems option.
