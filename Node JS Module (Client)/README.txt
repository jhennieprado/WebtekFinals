1. Install Node JS
   - download it from https://nodejs.org/en/
2. Put all the files in a new directory
3. Open Node JS and navigate to the new directory
4. Type npm -install
5  After the installation of the dependencies you can now run the server by
   - node index.js [PORT NUMBER] (eg. node index.js 9000)
   - The server will be located in localhost:[PORT]/ (eg. locahost:9000/)
6. You can terminate the server by pressing CTRL+C
