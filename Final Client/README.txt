NODE JS (CLIENT MODULE) SETUP

1. Copy the client folder to your desktop
2. Open Node Js console
3. Navigate to the copied client folder in your desktop using cd desktop/client
4. Install the necessary dependencies "npm install -g"
5. Run the module by "node index.js 90" (eg node index.js <PORT>)
6. You can now access the module through http://localhost:90/
7. The server can be terminated by pressing CTRL+C
